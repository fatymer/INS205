# 🎂 Foody Castle

**Foody Castle** is a modern, responsive, and user-friendly cake ordering web application developed as part of an academic assignment.

The website provides customers with an attractive online platform where they can explore different cake varieties, place cake orders, learn more about the bakery, and contact the business.

The project is built using **HTML5, CSS3, and JavaScript** and is containerized using **Docker with Nginx**.

---

## 🌟 Features

- 🏠 Beautiful and responsive homepage
- 🎂 Cake menu with different cake varieties
- 🛒 Cake ordering form
- 📝 Customer order details collection
- 📅 Delivery date selection
- 💳 Payment method selection
- 📖 About Us page
- 📞 Contact Us page
- 💎 Modern glassmorphism user interface
- 📱 Mobile-friendly responsive design
- ✨ Smooth animations and interactive buttons
- ⬆️ Scroll-to-top functionality
- ❤️ Interactive user experience
- 🐳 Docker containerization using Nginx

---

## 🛠 Technologies Used

- **HTML5** – Website structure
- **CSS3** – Styling, responsive design, and glassmorphism effects
- **JavaScript** – Interactivity and dynamic functionality
- **Docker** – Application containerization
- **Nginx** – Web server used to serve the website
- **GitHub** – Source code management and version control
- **Docker Hub** – Docker image hosting
